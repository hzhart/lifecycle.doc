# 文件上传

## 文件上传
`
POST /admin/storage/upload
`

<table>
    <tr>
        <th style="width:150px;">请求参数</th>
        <th>必须</th>
        <th>说明</th>
        <th>备注</th>
    </tr>
    <tr>
        <td>upfile</td>
        <td>是</td>
        <td>上传文件name</td>
        <td>-</td>
    </tr>
</table>

> 上传文件采用form-data格式, 允许".jpg", ".jpeg", ".png", ".gif", ".doc", ".docx", ".xls", ".xlsx", ".ppt", ".pptx", ".pdf", ".zip", ".rar" 等类型上传

> 返回值

<table>
    <tr>
        <th style="width:150px;">返回参数</th>
        <th>类型</th>
        <th>说明</th>
        <th>备注</th>
    </tr>
    <tr>
        <td>key</td>
        <td>string</td>
        <td>上传成功的文件key</td>
        <td>-</td>
    </tr>
    <tr>
        <td>url</td>
        <td>string</td>
        <td>文件预览url</td>
        <td>-</td>
    </tr>
</table>
