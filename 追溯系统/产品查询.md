# 追溯系统

## 按追溯码查看产品

```
GET /public/product/get-by-trace-code?traceCode=:traceCode
```

## 物料列表

```
GET /public/component/?ids=:ids
```

## 物料详情

```
GET /public/component/:id
```

## 物流信息

```
GET /public/ship/get-by-product-id?productId=:productId
```
